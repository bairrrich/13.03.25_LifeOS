# /speckit.plan

## Purpose
Create technical implementation plan.

## Instructions
1. Review the specification
2. Identify technical components
3. Define architecture and design
4. Break down into implementation steps
5. Identify risks and dependencies

## Output
Create `.specify/specs/<feature-name>/plan.md` with:
- Architecture overview
- Component design
- Data models
- API contracts
- Implementation phases
- Risk assessment
