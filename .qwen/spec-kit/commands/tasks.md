# /speckit.tasks

## Purpose
Generate detailed task list for implementation.

## Instructions
1. Review the implementation plan
2. Break down into atomic tasks
3. Estimate effort for each task
4. Define task dependencies
5. Prioritize tasks

## Output
Create `.specify/specs/<feature-name>/tasks.md` with:
- Task breakdown
- Effort estimates
- Dependencies
- Priority order
- Definition of done
