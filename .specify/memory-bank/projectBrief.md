# Project Brief

## Overview
LifeOS — персональная операционная система для управления жизнью, задачами, целями и привычками. Spec-driven проект с использованием GitHub Spec Kit для спецификации, планирования и реализации функциональности.

## Core Requirements
- **Spec-Driven Development**: Каждая функция начинается со спецификации в `.specify/specs/`
- **Memory Bank**: Контекст проекта сохраняется в `.specify/memory-bank/` для AI-агентов
- **Quality First**: TDD, интеграционные тесты, код ревью через spec-kit
- **Modular Architecture**: Компонентная архитектура с чёткими контрактами

## Goals
1. Создать единую систему управления личными задачами и целями
2. Обеспечить spec-driven workflow для разработки
3. Поддерживать актуальный контекст для AI-агентов (Qwen Code)
4. Документировать все архитектурные решения

## Scope

### In Scope
- Система задач и проектов
- Трекер привычек
- Управление целями (OKR-подобный подход)
- Интеграции с внешними сервисами
- Spec-kit workflow для разработки

### Out of Scope
- Мобильные приложения (веб-первый подход)
- Мультипользовательский режим (персональное использование)
- Сложные социальные функции

## Success Metrics
- Все функции имеют спецификацию до реализации
- Memory Bank обновляется после каждой сессии
- 80%+ покрытие тестами критического пути
- Время на онбординг AI-агента < 5 минут
