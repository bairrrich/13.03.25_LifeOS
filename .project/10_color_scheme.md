# Цветовая схема LifeOS

## Обзор

LifeOS использует цветовое пространство **OKLCH** для точной настройки цветов с учётом восприятия человеческим глазом.

### Почему OKLCH?

- **Перцептивная равномерность** — одинаковое изменение значений даёт одинаковое визуальное изменение
- **Учёт яркости** — L (lightness) от 0 (чёрный) до 1 (белый)
- **Насыщенность** — C (chroma) от 0 (серый) до 0.2+ (яркий)
- **Оттенок** — H (hue) 0-360 градусов

## Light Theme (Светлая тема)

### Основные цвета

| Переменная | Значение | Описание |
|------------|----------|----------|
| `--background` | `oklch(1 0 0)` | Чистый белый фон |
| `--foreground` | `oklch(0.21 0.01 250)` | Тёмно-серый текст с синим оттенком |
| `--card` | `oklch(1 0 0)` | Белые карточки |
| `--border` | `oklch(0.92 0.003 250)` | Светло-серые границы |

### Primary (Основной цвет бренда)

```css
--primary: oklch(0.55 0.12 260);  /* Мягкий синий */
--primary-hover: oklch(0.5 0.14 260);
--primary-foreground: oklch(1 0 0);  /* Белый текст */
```

**Почему этот оттенок:**
- L=0.55 — достаточно яркий для кнопок, но не режет глаза
- C=0.12 — умеренная насыщенность, профессиональный вид
- H=260 — синий с фиолетовым оттенком (доверие, стабильность)

### Semantic Colors

| Цвет | Значение | Использование |
|------|----------|---------------|
| Success | `oklch(0.62 0.14 150)` | Зелёный — успех, подтверждение |
| Warning | `oklch(0.72 0.12 75)` | Жёлто-оранжевый — предупреждение |
| Info | `oklch(0.65 0.1 240)` | Синий — информация |
| Error | `oklch(0.57 0.18 25)` | Красный — ошибки |

## Dark Theme (Тёмная тема)

### Основные цвета

| Переменная | Значение | Описание |
|------------|----------|----------|
| `--background` | `oklch(0.14 0.005 250)` | Тёмно-серый с синим оттенком |
| `--foreground` | `oklch(0.92 0.005 250)` | Светло-серый текст |
| `--card` | `oklch(0.18 0.006 250)` | Карточки на тон светлее фона |
| `--border` | `oklch(0.28 0.01 250)` | Видимые, но не яркие границы |

### Primary для тёмной темы

```css
--primary: oklch(0.65 0.14 260);  /* Более светлый */
--primary-hover: oklch(0.7 0.12 260);
--primary-foreground: oklch(0.2 0.01 260);  /* Тёмный текст */
```

**Адаптации для тёмной темы:**
- L увеличена с 0.55 до 0.65 — лучше видимость
- C увеличена с 0.12 до 0.14 — компенсация восприятия
- Текст тёмный для контраста

## High Contrast Theme (Контрастная тема)

Для пользователей с нарушениями зрения.

| Переменная | Значение |
|------------|----------|
| `--background` | `oklch(1 0 0)` |
| `--foreground` | `oklch(0 0 0)` |
| `--primary` | `oklch(0.4 0.15 260)` |
| `--border` | `oklch(0 0 0)` |

**Особенности:**
- Максимальный контраст (чёрный на белом)
- Увеличенная насыщенность primary
- Меньший radius скругления

## Рекомендации по использованию

### ✅ Правильно

```tsx
// Кнопка действия
<Button variant="default">Сохранить</Button>

// Второстепенное действие
<Button variant="secondary">Отмена</Button>

// Опасное действие
<Button variant="destructive">Удалить</Button>

// Текст статуса
<span className="text-success">Успешно</span>
<span className="text-warning">Внимание</span>
<span className="text-error">Ошибка</span>
```

### ❌ Неправильно

```tsx
// Не используйте hard-coded цвета
<div style={{ color: '#3b82f6' }}>Текст</div>

// Не переопределяйте семантические цвета
<div className="text-destructive">Успешно</div>
```

## Accessibility (Доступность)

### Контрастность

Все комбинации текста и фона соответствуют **WCAG 2.1 AA**:

| Комбинация | Контраст | Стандарт |
|------------|----------|----------|
| Foreground / Background | 12.5:1 | ✓ AAA |
| Primary / Primary-fg | 4.8:1 | ✓ AA |
| Muted-fg / Muted | 4.2:1 | ✓ AA |

### Focus States

```css
:focus-visible {
  outline: 2px solid var(--ring);
  outline-offset: 2px;
}
```

Видимая обводка для навигации с клавиатуры.

## Переходы между темами

Плавные переходы обеспечиваются CSS transitions:

```css
* {
  transition-property: background-color, border-color, color;
  transition-duration: 150ms;
  transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
}
```

## Расширение палитры

Для добавления новых цветов используйте OKLCH конвертер:

1. Выберите базовый оттенок (Hue)
2. Настройте яркость (Lightness) для темы
3. Настройте насыщенность (Chroma)

### Пример: новый акцентный цвет

```css
/* Фиолетовый акцент */
--accent-purple: oklch(0.6 0.15 280);
--accent-purple-hover: oklch(0.55 0.18 280);
--accent-purple-fg: oklch(1 0 0);
```
