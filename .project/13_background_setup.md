# Настройка фона (Background) для LifeOS

## Обзор

Фон настроен с учётом визуального комфорта и различия между темами.

---

## Light Theme (Светлая тема)

### Значения
```css
--background: oklch(0.985 0.002 250);  /* Белый с едва заметным синим оттенком */
--card: oklch(1 0 0);                   /* Чистый белый для карточек */
```

### Характеристики
- **L (Lightness):** 0.985 — очень светлый, но не чисто белый
- **C (Chroma):** 0.002 — минимальная насыщенность
- **H (Hue):** 250 — лёгкий синий оттенок

### Почему так
- Чисто белый (`oklch(1 0 0)`) режет глаза
- Лёгкий оттенок создаёт глубину
- Карточки чисто белые — создают контраст с фоном

---

## Dark Theme (Тёмная тема)

### Значения
```css
--background: oklch(0.18 0.008 250);  /* Глубокий тёмно-серый с синим */
--card: oklch(0.22 0.01 250);         /* Светлее для карточек */
```

### Характеристики
- **L (Lightness):** 0.18 — тёмный, но не чёрный
- **C (Chroma):** 0.008 — едва заметная насыщенность
- **H (Hue):** 250 — синий оттенок (консистентность с light)

### Почему так
- Чёрный (`oklch(0 0 0)`) слишком контрастный
- 0.18 — оптимальная яркость для длительной работы
- Карточки на 0.04 светлее — видимая иерархия
- Синий оттенок создаёт "глубину"

---

## High Contrast Theme

### Значения
```css
--background: oklch(1 0 0);  /* Чистый белый */
--card: oklch(1 0 0);        /* Чистый белый */
```

### Характеристики
- Максимальный контраст
- Чёрные границы (`--border: oklch(0 0 0)`)
- Для пользователей с нарушениями зрения

---

## Сравнение тем

| Тема | Background | Card | Контраст |
|------|------------|------|----------|
| Light | `oklch(0.985 0.002 250)` | `oklch(1 0 0)` | Низкий |
| Dark | `oklch(0.18 0.008 250)` | `oklch(0.22 0.01 250)` | Средний |
| High Contrast | `oklch(1 0 0)` | `oklch(1 0 0)` | Максимальный |

## Градиенты

Добавлены subtle градиенты для визуального интереса:

### Subtle Gradient
```css
.bg-gradient-subtle {
  background: linear-gradient(
    135deg,
    var(--background) 0%,
    var(--card) 100%
  );
}
```

### Primary Gradient
```css
.bg-gradient-primary {
  background: linear-gradient(
    135deg,
    var(--primary) 0%,
    var(--accent) 100%
  );
}
```

## Проверка

### 1. Откройте `/theme-check`
Проверьте что фон меняется при переключении тем.

### 2. Визуальная проверка
- **Light:** Фон слегка серовато-белый, карточки белые
- **Dark:** Фон тёмно-серый с синим оттенком, карточки светлее
- **High Contrast:** Чёрно-белая тема

### 3. DevTools
Проверьте вычисленные значения:
```javascript
getComputedStyle(document.documentElement)
  .getPropertyValue('--background')
```

## Рекомендации

### ✅ Используйте
- `bg-background` для основного фона
- `bg-card` для карточек и панелей
- `bg-secondary` для второстепенных элементов

### ❌ Избегайте
- Жёстко заданных цветов (`#ffffff`, `#000000`)
- Прямых значений OKLCH в компонентах
- Игнорирования семантических имён

---

## Расширение

Для добавления нового цвета фона:

1. Добавьте CSS переменную в `globals.css`
2. Экспортируйте в `@theme inline`
3. Используйте через `bg-[var(--your-variable)]`

Пример:
```css
:root {
  --background-muted: oklch(0.96 0.003 250);
}

@theme inline {
  --color-background-muted: var(--background-muted);
}
```
