# Решение проблемы с переключением тем

## Проблема
`data-theme` переключается в HTML, но визуально тема не меняется — везде одна (тёмная) тема.

## Причина
CSS переменные переопределялись неправильно. Светлая тема была определена на `:root`, и когда устанавливался `data-theme="dark"`, переменные из `:root` всё ещё применялись.

## Решение

### 1. Исправлена структура CSS

**Было (НЕПРАВИЛЬНО):**
```css
:root,
[data-theme="light"] {
  --background: oklch(1 0 0);
}

[data-theme="dark"] {
  --background: oklch(0.14 0 0);
}
```

Проблема: `:root` имеет более высокий приоритет и переопределяет `[data-theme="dark"]`.

**Стало (ПРАВИЛЬНО):**
```css
:root {
  /* Светлая тема по умолчанию */
  --background: oklch(1 0 0);
}

[data-theme="dark"] {
  /* Полное переопределение ВСЕХ переменных */
  --background: oklch(0.14 0 0);
  --foreground: oklch(0.92 0 0);
  /* ... все переменные */
}

[data-theme="light"] {
  /* Явное переопределение для light */
  --background: oklch(1 0 0);
  /* ... все переменные */
}
```

### 2. Добавлена страница визуального теста

Откройте `/theme-visual-test` для проверки:
- Все варианты кнопок
- StatCard компоненты
- Progress Bar
- Badge
- Cards

### 3. Проверка работы

1. **Откройте** `/theme-visual-test`
2. **Нажмите** кнопку "Dark" — фон должен стать тёмным
3. **Нажмите** кнопку "Light" — фон должен стать белым
4. **Нажмите** кнопку "High Contrast" — чёрно-белая тема

### 4. Быстрая проверка в DevTools

1. Откройте консоль браузера
2. Выполните: `document.documentElement.setAttribute('data-theme', 'dark')`
3. Проверьте, что фон стал тёмным
4. Выполните: `document.documentElement.setAttribute('data-theme', 'light')`
5. Проверьте, что фон стал белым

### 5. Проверка CSS переменных

В DevTools выполните:

```javascript
// Проверка текущих значений
getComputedStyle(document.documentElement).getPropertyValue('--background')
getComputedStyle(document.documentElement).getPropertyValue('--primary')
```

При переключении тем значения должны меняться.

## Проверка специфичности селекторов

Специфичность селекторов (от большей к меньшей):

1. `[data-theme="dark"]` — 0,1,0
2. `:root` — 0,1,0
3. `*` — 0,0,0

Когда `data-theme="dark"` установлен на `<html>`, селектор `[data-theme="dark"]` применяется и переопределяет `:root`.

## Частые проблемы

### Проблема 1: Тема не меняется
**Решение:** Проверьте, что `data-theme` атрибут установлен на `<html>` элементе

### Проблема 2: Некоторые цвета не меняются
**Решение:** Проверьте, что для этих цветов определены CSS переменные в нужной теме

### Проблема 3: Мигание при загрузке
**Решение:** Это нормально. next-themes загружает тему из localStorage после монтирования
