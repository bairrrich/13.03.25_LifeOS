# Suggested Commands

## Development
```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Production build
npm run build

# Start production server
npm run start

# Run linter
npm run lint
```

## Git & Commit
```bash
# Stage and commit
git add .
git commit -m "type: description"

# Commit types: feat, fix, docs, style, refactor, test, chore
```

## System Commands (Windows)
```bash
# Directory listing
dir /b

# File search
grep -r "pattern" .

# Find files
dir /s /b *.ts
```
